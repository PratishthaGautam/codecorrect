About Code Refactor

A 4th year major project focused on code quality improvement through intelligent refactoring and automated documentation.

Project Overview

Code Refactor is a software engineering tool designed to improve code quality and maintainability. The project combines static analysis with intelligent refactoring suggestions.

Our tool helps developers identify code issues and provides automated solutions for better software maintenance.

Quick Start

Installation

git clone https://github.com/team/code-refactor
npm install
npm start

Basic Usage

coderefactor analyze MyClass.java
coderefactor document src/
coderefactor refactor --pattern extract-method

Refactoring Patterns

Extract Method

Breaks down long methods into smaller functions.

Extract Class

Separates responsibilities into different classes.

Rename Variable

Improves code readability with better names.

Replace Conditional

Simplifies complex conditional logic.

Remove Duplication

Eliminates repeated code blocks.

Introduce Parameter

Makes methods more flexible and reusable.
